# Get Status Code - Python - Flask
Return the status code of the web request. This is implemented with flask and boostrap

## Prerequisites
- Python
- Flask
- Bootstrap

## Checkout & Build
```
* `git clone` https://github.com/vishnuar/get-status-code.git
* `cd` get-status-code
```

## Run the apps
```shell
python app.py
```
